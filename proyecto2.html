Proyecto 2 Analizador Lexico

El objetivo de este proyecto fue desarrollar un analizador léxico utilizando Java, una herramienta fundamental en el proceso de compilación que convierte una secuencia de caracteres en una secuencia de tokens significativos. El analizador léxico es el primer paso en la cadena de procesamiento del lenguaje de programación, y su función principal es identificar las unidades léxicas del código fuente, como palabras clave, identificadores, operadores y literales.

El proyecto resultó en un analizador léxico funcional que convierte de manera eficiente el código fuente en una secuencia de tokens, permitiendo un análisis más profundo en etapas posteriores del proceso de compilación. La implementación en Java demostró la capacidad del lenguaje para manejar el análisis léxico y facilitar la creación de herramientas de procesamiento de lenguaje.
Este analizador léxico puede ser utilizado como parte de un compilador o intérprete para lenguajes de programación personalizados, así como para herramientas de análisis estático de código y editores de texto avanzados que necesitan procesar y entender el código fuente.
