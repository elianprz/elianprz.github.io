Proyecto 5 Juego de Ajedrez

Este proyecto consiste en el desarrollo de un juego de ajedrez utilizando Java, que ofrece una experiencia completa de juego de ajedrez en una interfaz gráfica. El objetivo principal fue crear un juego de ajedrez funcional que permita a los jugadores jugar contra otros usuarios o contra una inteligencia artificial (IA), proporcionando una experiencia de usuario fluida y visualmente atractiva

El juego de ajedrez en Java proporciona una experiencia completa de ajedrez con una interfaz gráfica amigable y una lógica de juego robusta. Los jugadores pueden disfrutar de partidas tanto contra la IA como contra otros jugadores, con una variedad de opciones de configuración y personalización. El proyecto demuestra la capacidad de Java para crear aplicaciones interactivas y gráficas complejas.
