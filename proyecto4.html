Proyecto 4 Programa Interbloqueo

Este proyecto consiste en el desarrollo de un programa en C# que gestiona el interbloqueo de procesos, permite la pausa y detención de procesos activos, y proporciona información sobre las redes activas en el sistema. El objetivo principal es crear una herramienta robusta para la administración y el monitoreo de procesos en un entorno de Windows, así como para la supervisión de la actividad de red.

El programa proporciona una solución completa para la gestión de interbloqueo, control de procesos, y monitorización de redes en sistemas Windows. Con una interfaz de usuario clara y funcionalidades robustas, el software facilita la administración eficiente de los recursos del sistema y ofrece herramientas valiosas para la supervisión de la actividad de red y procesos.
